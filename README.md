# zihan.com.au

Source for [www.zihan.com.au](https://www.zihan.com.au), the personal academic homepage of Zihan Wang (Postdoctoral Research Fellow, University of Queensland).

## Build locally

```sh
bundle install
bundle exec jekyll serve
```

Open `http://localhost:4000/`.

## Deploy

Pushes to `main` deploy automatically via GitHub Pages. The custom domain is set in [`CNAME`](CNAME).
